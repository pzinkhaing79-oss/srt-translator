# 🎥 AI Subtitle Translator (Gemini Powered)

ဒီ App ဟာ SRT Subtitle ဖိုင်တွေကို AI (Gemini 1.5 Flash/Pro) အသုံးပြုပြီး မြန်မာဘာသာသို့ အလိုအလျောက် ဘာသာပြန်ပေးတဲ့ Web Application ဖြစ်ပါတယ်။

## ✨ Features
- **Multi-Model Support:** Gemini 1.5 Flash နဲ့ Pro နှစ်မျိုးလုံး သုံးလို့ရပါတယ်။
- **SRT Preservation:** အချိန်မှတ် (Timestamps) တွေကို မထိခိုက်ဘဲ စာသားကိုပဲ တိကျစွာ ပြန်ပေးပါတယ်။
- **User API Key:** အသုံးပြုသူရဲ့ ကိုယ်ပိုင် API Key ကို သုံးနိုင်လို့ ကန့်သတ်ချက်နည်းပါတယ်။

## 🚀 Tech Stack
- **Frontend:** Next.js, Tailwind CSS
- **Backend:** Python FastAPI
- **AI:** Google Gemini API

## 🛠️ How to Setup
1. Clone the repo: `git clone [your-repo-link]`
2. Install Backend dependencies: `pip install -r requirements.txt`
3. Run Backend: `uvicorn main:app --reload`
4. Run Frontend: `npm run dev`
